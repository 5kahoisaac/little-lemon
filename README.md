# Little Lemon Reserve a Table

A modern React/Vite implementation of the Little Lemon mobile reserve-a-table feature. The UI follows the Little Lemon course mockup style: dark green brand hero, yellow CTA buttons, rounded cards, mobile-first layout, reservation forms, and a confirmation screen.

## Grading Checklist Coverage

- Matches the Little Lemon reservation UI/UX mood and wireframe flow
- Uses semantic HTML: `main`, `header`, `nav`, `section`, `article`, `figure`, `fieldset`, `legend`, and `aside`
- Includes accessible form labels, `aria-invalid`, `aria-describedby`, alerts, focus states, and `aria-live`
- Fully functional booking form with validation
- Responsive for mobile and desktop
- Includes unit tests with Vitest and React Testing Library
- Clear project structure and documentation

## Setup
